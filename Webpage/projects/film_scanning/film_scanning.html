35mm Film digitizing

Ever since shooting Film, ive wanted to be able to scan my negatives myself.
Scanning them myself has many advantages, for example beeing able to set how the colors look or fixing white-balance and many more things
which you couldn't do as good when getting your scans back as a jpg from a lab.

Equipment

Scanning rig

To digitize my 35mm Film ive built a little contraption to scan Film with my DSLR.
The first thing i needed was some type of holder for the film.
So i looked around for some holder for film on thingiverse.
i found this rig here, which i quite liked.


Picture belongs to ondrejp_sk


Lens

In terms of lens choice, i went with a nikon 55mm AI-s f3.5.







Reproduction ratio

The lens is great for macro use already, but a reproduction ratio of 1:1, fully covers the full-frame sensor of a full frame sensor like my nikon D750.
To calculate the reproduction ratio, i used this calculator here.

Now i knew which extension tube length i needed. But to find a set of extension tubes that add up to exactly 48mm proved to be quite the challenge.
So instead of buying a set of extension tube, i bought macro bellows off ebay. Theese can be set to any length you desire.




Light source

As a light source, i wanted to use my old mobile phone.
I had the phone holder piece from a selfie stick laying around so i used that as a holder. Since it has a 1/4"-20 thread nut in the bottom, it can be used to mount it to a tripod.
